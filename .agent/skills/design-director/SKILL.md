description: Use this skill when the user asks to redesign UI, improve layout hierarchy, spacing, or make the site more institutional and premium.

Design Director: Institutional Premium

Goal

I transform the UI into an institutional, high-authority system with strong hierarchy and restraint.

Non-negotiables

Use semantic tokens only.

Use one accent color for interactive states.

Reduce visual noise.

Maintain consistent patterns across headings, sections, and buttons.

Do not introduce decorative elements without conversion purpose.

Process

Diagnose 7 issues with file paths.

Propose 2 improvement directions.

Select 1 and explain why.

Implement minimal safe changes.

Verify hierarchy, spacing, and CTA clarity.

Output

Issues

Chosen direction

Files modified
