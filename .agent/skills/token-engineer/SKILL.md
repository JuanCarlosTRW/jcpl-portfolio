Token Engineer: Semantic Design System

Goal

Create a semantic Tailwind token system and eliminate hardcoded hex values.

Rules

Tailwind config is the source of truth.

No raw hex inside components.

Tokens must be semantic, not visual names.

Process

Extract existing colors.

Define minimal semantic tokens.

Refactor project to use tokens.

Verify hover and focus states.

Output

Token map

Updated config

Files changed
